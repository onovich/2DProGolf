# 2D Pro Golf 序章与第一城市内容文档

## 1. 文档目标

这份文档用于定义首个可玩纵切片的内容范围，覆盖以下部分：

1. 序章开场。
2. 精灵向导角色。
3. 第一城市“雾岸练习镇”。
4. 教学赛事结构。
5. 首批 NPC 与对白方向。

本阶段目标不是把世界观一次讲完，而是让玩家在 20 到 40 分钟内完成“进入异世界、学会击球、认识向导、打完基础训练赛、理解要去更大世界参赛”的流程。

## 2. 序章整体结构

### 2.1 体验目标

1. 让玩家快速理解主角处境。
2. 让玩家把“学习高尔夫”当成眼前最具体的生存任务。
3. 让精灵向导自然介入，而不是纯教程播报器。
4. 让玩家在教程结束时明确知道后面会去城市巡回参赛。

### 2.2 序章节奏

建议切成 5 个段落：

1. 事故余响。
2. 异世界初醒。
3. 精灵接触。
4. 教学球场。
5. 通往世界地图的开门时刻。

## 3. 序章剧情流程

### 3.1 段落一：事故余响

演出形式：黑屏、少量字幕、音效、抽象图形。

表现建议：

1. 不直接画车祸具体过程。
2. 用车窗倒影、导航播报、刹车摩擦、短促白光去暗示。
3. 画面随后转成一个夸张的高尔夫球飞行抛物线，作为转场意象。

字幕建议：

1. 今天本来只是普通加班日。
2. 你只是想早点回家。
3. 然后有东西撞了上来。
4. 或者是你撞上了什么。
5. 总之，之后就只剩一声很像开球的响动。

### 3.2 段落二：异世界初醒

主角在草地上醒来，看到远处旗杆和陌生天空。

目标：

1. 让玩家进入第一段可控移动或镜头观察。
2. 把旗杆作为视觉焦点。
3. 给出“这个世界以高尔夫为秩序”的第一印象。

环境文案建议：

1. 远处能看到像城镇、又像练习场的建筑轮廓。
2. 草地边有奇怪的告示牌，写着“请遵守击球礼仪”。
3. 天空里有缓慢漂浮的发光线条，像风向，也像某种规则本身。

### 3.3 段落三：精灵接触

精灵出现方式建议：先是声音，后是一个发光的小型几何体逐渐聚拢成形。

精灵第一句建议偏轻巧，不要过于神秘：

“醒了？太好了。我刚还在想，是先教你握杆，还是先教你接受人生已经彻底跑偏了。”

这段目标：

1. 快速交代这里是高尔夫异世界。
2. 交代主角暂时回不去。
3. 给出明确短期目标：先学会打球，先活下来，先赢最基础的比赛。

### 3.4 段落四：教学球场

精灵带主角前往雾岸练习镇边缘的小球场。

这里不需要大段叙事，重点是每场教学赛都像是剧情推进的一部分。

### 3.5 段落五：通往世界地图的开门时刻

当玩家完成最后一场教程赛后，镇上的赛事管理员交给主角一张“巡回证”。

精灵告诉玩家：

1. 世界很大。
2. 不是每个人都想回去。
3. 但如果你真想回去，就得一直赢下去。

这一段结束时，玩家第一次看到更大的世界地图界面或其占位版。

## 4. 教学赛事设计

建议共 5 个教学关卡。

### 4.1 教学关卡 1：把球打出去

目标：学会瞄准、拖拽、击球。

球场特征：

1. 极短直线洞。
2. 无风。
3. 无沙坑、无水域。
4. 超宽球道。

精灵提示重点：

1. 先别想太多，先把球打出去。
2. 在这里，能把球从 A 点弄到 B 点，已经比很多人有规划能力了。

### 4.2 教学关卡 2：理解球杆差异

目标：理解木杆、铁杆、挖起杆、推杆的功能区别。

球场特征：

1. 两段式短洞。
2. 中间有轻微长草区。
3. 无明显风。

精灵提示重点：

1. 不是每一杆都靠大力解决。
2. 你在原来世界如果也是这么想的，难怪会出事。

### 4.3 教学关卡 3：理解风

目标：学习顺风、逆风、侧风的最基础判断。

球场特征：

1. 低强度侧风或轻逆风。
2. 仍然保持较宽容球道。
3. 禁止叠加复杂坡度。

精灵提示重点：

1. 风不是你的敌人，是考试题。
2. 只是这边的考试题比较喜欢刁难刚醒的人。

### 4.4 教学关卡 4：理解坡度与推杆

目标：学会看果岭坡度，理解推杆为什么会偏。

球场特征：

1. 超短洞。
2. 明确可读的果岭坡度。
3. 自动切换推杆或强提示使用推杆。

精灵提示重点：

1. 球不会照顾你的情绪，它只会往低处走。
2. 所以你最好比它更早看出低处在哪。

### 4.5 教学关卡 5：小型正式赛

目标：综合运用前面学到的规则，并完成第一次“正式获胜”。

球场特征：

1. 3 洞练习赛。
2. 有轻风、轻坡度、少量沙坑。
3. 有一名新手对手作为陪练。

完成奖励：

1. 获得巡回证。
2. 解锁第一城市主界面。
3. 解锁世界地图预览。

## 5. 精灵向导设定

### 5.1 角色定位

精灵是功能向导、吐槽役、规则翻译器，也是主角在异世界的第一位同伴。

### 5.2 性格关键词

1. 话多。
2. 轻快。
3. 偶尔刻薄。
4. 实际可靠。
5. 对这个世界的荒谬习以为常。

### 5.3 角色功能

1. 教程提示。
2. 赛事复盘。
3. 剧情推进。
4. 情绪缓冲。
5. 替玩家说出一部分吐槽。

### 5.4 视觉表现建议

因为当前美术约束以纯代码方案为主，精灵建议用以下方式表现：

1. 以发光小球加几何翅片作为主体。
2. 表情用简单 SVG 眼睛和嘴形切换。
3. 情绪变化通过颜色、粒子和运动轨迹表达。
4. 对话头像可使用圆形徽记加少量表情变化。

### 5.5 语言风格规则

1. 句子短。
2. 吐槽多。
3. 不说长篇大论设定。
4. 经常把高尔夫规则讲得像生活建议。
5. 不做过度卖萌。

### 5.6 首批台词示例

1. “先别急着绝望，很多人第一次来这儿的时候，连球和石头都分不清。”
2. “你现在需要的不是答案，是一支能把球打起来的杆。”
3. “逆风的时候别跟风赌气，风最喜欢跟人赌气。”
4. “你这杆思路没错，错在你把勇敢和乱来混在一起了。”
5. “恭喜，你已经学会了这个世界最基础的礼貌：把球往洞的方向送。”

## 6. 第一城市：雾岸练习镇

### 6.1 城市定位

雾岸练习镇是所有新来者最常抵达的过渡区域，也是最宽容的球场聚落。它既像练习场，也像疗养地，还带一点临时收容所的味道。

### 6.2 城市功能

1. 教学与新手赛事。
2. 初级商店与训练场。
3. 初始 NPC 关系建立。
4. 世界地图第一层解锁。

### 6.3 视觉与氛围

视觉关键词：晨雾、低饱和青绿、木质牌面、半透明风线、安静但不空旷。

程序化表现建议：

1. 地面色相较柔和，风线粒子较轻。
2. 建筑物以简单几何块面和招牌 SVG 组合表现。
3. 赛事公告牌、练习旗杆、长椅、热饮摊可作为主要环境点缀。

### 6.4 城市节点建议

1. 镇入口。
2. 初级练习场。
3. 三洞训练区。
4. 小型赛事登记处。
5. 热饮摊。
6. 装备修整铺。

### 6.5 第一城市的玩法重点

1. 低风环境。
2. 短洞为主。
3. 宽球道。
4. 少量长草和沙坑。
5. 强调“理解规则”，而不是“拼极限操作”。

## 7. 首批 NPC 设计

建议首批放入 8 名具名 NPC，加 3 名球场路人观众。

### 7.1 岳止

身份：赛事登记员，前互联网产品经理。

角色功能：教程后第一个正式发任务的人。

性格：冷静、疲惫、流程感极强。

打球风格：稳健型。

对白示例：

1. “欢迎。别紧张，这里的表不用你填太多，毕竟你多半也不知道自己怎么来的。”
2. “如果你想回去，先从赢最不丢人的比赛开始。”
3. “规则我可以讲一遍，但风不会因为你听懂了就变温柔。”

### 7.2 乔美岚

身份：热饮摊老板，前连锁奶茶店店长。

角色功能：轻松闲聊、提供休息感。

性格：嘴快、爱评价人。

打球风格：激进型。

对白示例：

1. “我以前一天要哄两百个客人，现在只要看你们和风吵架，轻松多了。”
2. “你这杆啊，气势有，结果像打给昨天的自己看的。”

### 7.3 孟既白

身份：装备修整师，前机械工程师。

角色功能：球杆说明、后续装备系统入口。

性格：严谨、容易讲太细。

打球风格：推进型。

对白示例：

1. “球杆不是玄学。至少大部分时候不是。”
2. “你要是愿意，我可以用比你人生规划还清楚的方式给你讲清楚杆面角度。”

### 7.4 唐栖

身份：练习场常驻选手，前插画师。

角色功能：第一位可重复切磋的年轻对手。

性格：散漫、想法飘，但观察力好。

打球风格：稳健型，和人设形成反差。

对白示例：

1. “我以前总以为自己讨厌规则，后来发现我只是讨厌 Excel。”
2. “这里的风看着乱，其实比人好懂。”

### 7.5 马会南

身份：保安兼球场维护员，前工地班组长。

角色功能：边界提醒、礼仪说明、场地世界观补充。

性格：直来直去，但不凶。

打球风格：短杆型。

对白示例：

1. “别把球打出边界，捡球的是我，不是神。”
2. “你别看我说话糙，我推杆可细。”

### 7.6 沈遥

身份：练习赛陪练，前中学语文老师。

角色功能：教学关卡 5 的主要对手。

性格：温和、会拐着弯教育人。

打球风格：稳健型。

对白示例：

1. “别急着赢，先弄明白你这一杆想表达什么。”
2. “有些球要直接打，有些球得先想清楚退路。”

### 7.7 韩卯

身份：刚到不久的新人，前销售。

角色功能：与主角形成镜像，承担新手共鸣。

性格：嘴硬、焦虑、行动快。

打球风格：激进型。

对白示例：

1. “我知道要稳，但我一稳就觉得人生又回去了。”
2. “我每次都想一杆解决问题，问题通常不会同意。”

### 7.8 林稚禾

身份：雾岸镇临时主持人，前电台主播。

角色功能：赛事播报、结算氛围、章节开场提示。

性格：职业化、声音温柔、私下很碎念。

打球风格：情绪型。

对白示例：

1. “欢迎来到今天的练习赛。愿各位都能在风里保住一点体面。”
2. “你刚才那杆如果算艺术，我勉强愿意当观众。”

## 8. 球场观众 NPC

观众对话应短，不打断节奏，触发条件建议为球落点靠近或回合间短暂停留。

### 8.1 老洪

身份：前出租车司机。

对白示例：

1. “我一看你这球路，就知道你方向感平时也不太好。”

### 8.2 阿澄

身份：前实验室助理。

对白示例：

1. “风和坡都没错，错的是你以为自己已经把它们都算进去了。”

### 8.3 许萄

身份：前婚礼摄影师。

对白示例：

1. “刚才那杆很好看。可惜这里只有成绩，不给构图分。”

## 9. 首个可玩版本的内容需求

### 9.1 必做内容

1. 序章开场字幕与转场。
2. 精灵基础台词 30 到 40 句。
3. 雾岸练习镇基础节点。
4. 5 个教学关卡。
5. 8 名具名 NPC 的首次对话与复用短对白。
6. 1 场 3 洞正式练习赛。

### 9.2 够用即可内容

1. 城镇自由移动先做简化界面跳转，不强求大地图走动。
2. NPC 立绘先用几何头像加色彩区分。
3. 观众互动先做文本泡泡，不强求复杂演出。

## 10. 收尾目标

当玩家完成这段内容后，应该明确感受到三件事：

1. 这个世界是荒诞但有规则的。
2. 高尔夫是我必须学会的生存语言。
3. 如果想回地球，我就得继续打下去。
